Open your terminal in this folder and run:

```bash

npm install

# 2. Run
npm run dev

# 3. Open browser to the URL shown (usually http://localhost:5173)
```

## 🎓 New to React/Vite?

**Don't worry!** You don't need to know React to customize this website.

### To change text:
- Just edit the words in the component files
- Find the text in quotes and replace it

### To change images:
- Put your images in the `public/assets/` folder
- Update the image path in the code

### To change colors:
- Update the color values in `src/index.css`
- All colors are in one place

**Full instructions in CUSTOMIZATION_GUIDE.md**
