# Tidyzon - Professional Service Platform

A modern, responsive website for Tidyzon - your trusted service partner for home and car cleaning needs.

## 🚀 Features

- **Modern Design**: Clean, professional interface with smooth animations
- **Responsive**: Fully optimized for desktop, tablet, and mobile devices
- **Fast Performance**: Built with React and Vite for lightning-fast loading
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Accessible**: ARIA labels and keyboard navigation support

## 🎨 Design Elements

### Color Palette
- **Primary Green**: #2d5016 - Main brand color
- **Secondary Green**: #4a7c59 - Supporting elements
- **Accent Green**: #6bb77b - Highlights and CTAs
- **Light Green**: #a8d5a8 - Subtle accents

### Sections
1. **Header** - Navigation with mobile menu support
2. **Hero** - Eye-catching introduction with CTA
3. **Welcome** - Company overview and mission
4. **Services** - Mobile car cleaning and trash bin cleaning
5. **Features** - Key platform benefits
6. **Testimonials** - Customer reviews and ratings
7. **CTA** - Final call-to-action
8. **Footer** - Company information and links

## 🛠️ Tech Stack

- **React 18** - Modern UI library
- **Vite** - Next-generation frontend tooling
- **Lucide React** - Beautiful, consistent icons
- **CSS3** - Modern styling with animations
- **Google Fonts** - Inter font family

## 📦 Installation

1. Navigate to the project directory:
```bash
cd tidyzon-website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit the local URL shown in the terminal (typically `http://localhost:5173`)

## 🏗️ Build for Production

Create an optimized production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
